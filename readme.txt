# AI Business Intelligence Copilot

## Overview

An AI-powered Business Intelligence assistant that converts natural language questions into SQL queries, executes them on a database, and presents results with visualizations.

## Features

- Natural Language to SQL using Gemini
- Dynamic Schema Discovery
- SQLite Database Integration
- Interactive Streamlit Dashboard
- Automatic Data Visualizations
- Business Insights from Natural Language Queries

## Tech Stack

- Python
- SQLite
- Pandas
- Streamlit
- Google Gemini API

## Architecture

User Question
↓
Gemini AI
↓
SQL Query Generation
↓
SQLite Database
↓
Results
↓
Charts & Insights

## Example Questions

- How many orders are there?
- Show order status distribution
- Top 5 states by revenue
- Show monthly order trend

## Dataset

Olist Brazilian E-Commerce Dataset (100k+ orders)
